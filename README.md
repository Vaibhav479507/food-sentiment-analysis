# 🍽️ Food Sentiment Analysis

An AI-powered Machine Learning web application that predicts whether a restaurant review is **Positive** or **Negative** using Natural Language Processing (NLP).

## 🚀 Live Demo
👉 [Open App](https://food-sentiment-analysis.streamlit.app/)

## 📌 Features

- Predict sentiment for a single restaurant review
- Bulk prediction using CSV files
- Confidence score for each prediction
- Interactive Streamlit web interface

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## 💡 How It Works

1. Enter a restaurant review.
2. Click **Predict**.
3. The trained ML model analyzes the review.
4. The app displays:
   - Positive or Negative sentiment
   - Confidence score
