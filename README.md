# 🍽️ Food Sentiment Analysis

An AI-powered Machine Learning web application that predicts whether a restaurant review is Positive or Negative.

## 🚀 Live Demo
👉 [Open App](https://food-sentiment-analysis.streamlit.app/)

## 📌 Features

- Predict sentiment for a single review
- Bulk prediction using CSV
- Confidence score
- Streamlit Web App
